Для того чтобы развернуть у себя приложение "Социальная сеть", Вам потребуется:
  1. Ruby 2.3.1p112 (2016-04-26 revision 54768)
  2. Фреймворк Rails
    Для установки Rails используйте команду gem install.
  3. Установить все пакеты для того, чтобы развернуть приложение.
    Выполните команду bundle.
  4. После того, как все пакеты установились, выполнить следующую команду:
    rake db:create db:migrate db:seed
  5. После чего можете запускать приложение с помощью команды.
    be rails s
